## Used technologies:

- Frontend: React Js -> Redux, Redux Thunk, Axios, React Router v-6, Material UI, Yup, Formik, Helmet (for SEO)
- BackEnd: Node JS (Express JS), JWT token, Multer, Express Validator, Bcrypt JS
- Git: GitHub
- Hosting: Netlify
